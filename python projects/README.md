## 🐍 Python (CIS 261: Introduction to Programming Using Python)

This folder contains Python projects completed as part of CIS 261: Introduction to Programming Using Python. The work demonstrates foundational programming concepts including control flow, functions, object-oriented programming, data handling, and user interaction through console-based applications.

---

🃏 [Deck of Cards](https://github.com/olivialebron/Deck-of-Cards)  
  Simulates a standard deck of cards using object-oriented programming. Demonstrates class design, randomization, and basic game logic.

📐 [Rectangle](https://github.com/olivialebron/Rectangle)  
  Calculates properties of a rectangle such as area and perimeter. Highlights use of functions, user input, and basic arithmetic operations.

🌍 [Country](https://github.com/olivialebron/Country)  
  Models country-related data using classes. Demonstrates object-oriented concepts including attributes and methods.

🎬 [Movie Guide Part 1](https://github.com/olivialebron/MovieGuidePart1-)  
  Introduces a movie management system using lists and user input. Focuses on control flow and basic data structures.

🎬 [Movie Guide Part 2](https://github.com/olivialebron/MovieGuidePart2)  
  Expands on Part 1 by adding enhanced functionality and improved data handling. Demonstrates program scalability and modular design.

🧾 [Invoice Line Item Calculator](https://github.com/olivialebron/InvoiceLineItemCalculator)  
  Computes invoice totals with multiple inputs. Demonstrates functions, loops, and input validation.

🎮 [Guessing Game](https://github.com/olivialebron/Guessing-Game)  
  Interactive number guessing game using loops and conditionals. Highlights control flow and user interaction.
