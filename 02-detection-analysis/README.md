## 📡 2. Detection & Analysis (Tier 1 / Tier 2 SOC)

This phase focuses on identifying suspicious activity, analyzing security events, and determining whether alerts represent real threats. It involves continuous monitoring, alert triage, and investigation using SIEM tools and system logs.

---

### 🧪 Platforms & Environments

Hands-on experience was developed using industry-relevant cybersecurity platforms:

- Blue Team Labs Online (BTLO): Real-world SOC investigation scenarios  
- TryHackMe: SOC Level 1 training and log analysis labs  
- Splunk: SIEM platform for log ingestion, correlation, and investigation  
- Windows Event Logs & Sysmon: Endpoint activity monitoring and analysis  
- Wireshark: Network traffic inspection and packet analysis  

---

### ✅ Demonstrated Skills

#### 🔍 Log Analysis & SIEM Investigation
- Queried and analyzed logs using Splunk (SPL)  
- Correlated events across multiple data sources  
- Identified anomalies in authentication and system activity  

#### 🚨 Alert Triage
- Distinguished between false positives and true security incidents  
- Investigated alerts based on severity and context  
- Prioritized incidents for escalation  

#### 🧬 Threat Detection & IOC Identification
- Identified indicators of compromise (IP addresses, domains, hashes)  
- Detected brute force attacks, phishing attempts, and malware activity  
- Recognized suspicious patterns in logs and network traffic  

#### 🧠 Threat Analysis & Contextualization
- Analyzed attacker behavior using MITRE ATT&CK concepts  
- Investigated user activity, system processes, and network connections  
- Determined scope and potential impact of incidents
